Tebrikler, görev içeriğine ulaştın!
İşte beklediğimiz görevin gereksinimleri;
Tercih ettiğiniz bir arka uç dili (tercihen Java / Go / PHP / Python) ve ön yüz JS çatısı ile (tercihen React / Vue / Angular)
Temel seviyede kullanıcı üyeliği yapılabilen, oturum açma ve kapatma özelliklerine sahip bir web projesi geliştirilmesi beklenmektedir.
Oturum açan kullanıcılar, anasayfa üzerinde 3 adet ürün görmelidir. Bu ürünler dinamik veya statik olarak sunulabilir. Oturum açmamış kullanıcılar ise oturum açmaya yönlendirilmelidir.
Bu ürünlere canlı olarak açık artırma yapılıp, teklif verilebilmelidir. Bir başka kullanıcı, canlı olarak bu açık artırmayı takip edip, sayfayı yenilemeden güncel fiyatı görebilmelidir.
Oturum bilgileri için Redis kullanılmalıdır. RDBMS veya NoSQL tercihi isteğe göre yapılabilir.
Notlar;
* Arka uç tercihe göre micro service / monolithic mimaride geliştirilebilir.
* Temel olarak beklenen özellikleri yerine getiren projede, tasarım ve ek özellikler için hayal gücünüzü işin içine dahil edebilirsiniz.
* Uygulamanın ve gerekiyorsa arayüzün Docker ile nasıl derlenip çalışır hale getirileceği README olarak yazılmış olmalıdır. docker-compose üzerinden tek bir komut ile çalışacak şekilde servis edilmelidir. Veri tabanı gibi bağımlılıklar da içerisinde yer almalıdır.
* Görev içeriğine ulaşmak için kullanılan kod ve proje GitHub'da özel bir kod deposuna gönderilerek, erişim izni insankaynaklari@kartaca.com'a verilmelidir. E-posta ile bağlantı adresi insankaynaklari@kartaca.com'a iletilmelidir. Kod deposu üzerinden paylaşım sağlanamıyorsa, indirilebilir dosyalara ulaşmamız için bir dosya/bağlantı iletilmelidir.
* README içerisinde proje kullanımı açıkca anlatılmalıdır.
* Kod deposuna, içeriğe ulaşmak için kullandığın registrationKey'i README içine eklenmelidir.
